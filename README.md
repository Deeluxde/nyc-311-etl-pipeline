NYC 311 ETL Pipeline
End-to-End ETL Pipeline Using Python, Pandas, SQLite, SQL, and Git
Project Overview

This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline built using Python to process real-world NYC 311 Service Request data.

The pipeline:

Extracts raw service request data from NYC Open Data
Cleans and standardizes the dataset using Pandas
Loads the transformed data into a SQLite relational database
Executes SQL queries to validate the data and support analytical reporting
Uses Git for version control and project management

The project follows data engineering best practices for building reproducible ETL workflows.
